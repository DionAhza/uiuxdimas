Serverpoint - Redesign Website

ServerPoint is a provider of web hosting, cloud VPS, and bare metal dedicated servers that has been operating for decades, providing infrastructure solutions for global websites and applications. Bare metal dedicated servers are physical server services where all resources are dedicated to a single user, providing high performance, full control, and scalability without the “shared resource” effect. In this project, the focus of the redesign is on ServerPoint’s Bare Metal Dedicated Servers page which introduces the bare metal infrastructure service, its functions, main features, and its benefits for business users or developers who need high-performance servers.

  • Role : Web Design
  • Scope : UI Design, Web Design
  • Date : February 2025
